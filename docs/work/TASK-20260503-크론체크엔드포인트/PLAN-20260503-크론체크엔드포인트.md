# Phase 4 — 크론 엔드포인트 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `POST /api/cron/check` 크론 엔드포인트를 구현해 PM2.5 임계값(35 μg/m³) 이하 전환 시 Web Push 알림을 발송한다.

**Architecture:** Redis에서 구독 목록을 로드하고 stationName 기준으로 그룹핑한 뒤, 각 측정소의 에어코리아 API를 `Promise.allSettled`로 병렬 호출한다. 이전값 > 35 → 현재값 ≤ 35 전환 시 구독자에게 push를 발송하고, 410 Gone 응답이면 해당 구독을 Redis에서 자동 삭제한다.

**Tech Stack:** Next.js App Router route handler, @upstash/redis, web-push, Jest

import { redis, KEYS } from '@/lib/redis';
import type { StoredSubscription } from '@/lib/webpush';

  const authHeader = request.headers.get('authorization');

  if (!token || token !== process.env.CRON_SECRET) {

    ((await redis.hgetall(KEYS.subscriptions)) as Record<string, string> | null) ?? {};

      if (!stationMap.has(sub.stationName)) stationMap.set(sub.stationName, []);
      stationMap.get(sub.stationName)!.push({ key, sub });
    } catch {
      // 파싱 실패 시 무시
    }
  }

  const stats = { processed: 0, notified: 0, deleted: 0, skipped: 0 };

  return NextResponse.json(stats);

            { title: '환기하기 좋은 날씨예요!', body: `${sub.regionName} PM2.5 ${pm25Value} μg/m³ — 지금 환기하세요.` },
          );
          stats.notified++;
        } catch (err: unknown) {
          if (
            err !== null &&
            typeof err === 'object' &&
            'statusCode' in err &&
            (err as { statusCode: number }).statusCode === 410
          ) {
            await redis.hdel(KEYS.subscriptions, key);
            stats.deleted++;
          }
        }
      }
    }

    await redis.set(KEYS.lastPm25(stationName), pm25Value);
  }

  return NextResponse.json(stats);
}
