## 작업 목표
cron-job.org가 30분 간격으로 호출하는 `POST /api/cron/check` 엔드포인트를 구현한다.

## 완료된 작업

### 4.1 `app/api/cron/check/route.ts`
- Bearer 토큰 인증 (CRON_SECRET)
- Redis HGETALL로 구독 목록 로드
- stationName 기준 Map 그룹핑
- Promise.allSettled로 병렬 API 호출
- prev > 35 AND current ≤ 35 전환 시 Web Push 발송
- 410 Gone 응답 시 구독 자동 삭제
- pm25Value null 시 last_pm25 갱신 생략
- 응답: `{ processed, notified, deleted, skipped }`

### 4.2 `__tests__/api/cron/check.test.ts`
- `@jest-environment node` 설정 (NextRequest Web API 지원)
- 10개 TDD 시나리오 전체 통과
