# TASK-20260517 — PWA 강화 (Phase 7)

## 연관 문서

## 작업 목표

Android/iOS에서 홈 화면 설치를 유도하고, iOS Web Push 알림이 정상 동작하도록 PWA를 강화한다.
스토어 배포 없이 무료로 "앱처럼" 쓰는 경험을 제공한다.

## 작업 항목

### 7.1–7.2 아이콘 + manifest
- sharp 스크립트로 192px, 512px PNG + Apple Touch Icon(180px) 생성
- `manifest.ts`에 아이콘 목록, `display_override` 추가

### 7.3 `useInstallPrompt` 훅 (TDD)
- 플랫폼 감지 (iOS / Android / other)
- `beforeinstallprompt` 이벤트 캡처 (Android)
- standalone 설치 상태 감지
- iOS: `promptInstall()` 호출 시 안내 모달 표시

### 7.4 `InstallPrompt` 컴포넌트 (TDD)
- Android: 하단 배너 + "앱으로 설치" 버튼
- iOS: 하단 배너 + "홈 화면에 추가" 3단계 안내 모달
- 설치 완료 시 자동 숨김

### 7.5–7.6 iOS Web Push 지원
- `useNotification`에 `requiresInstall` 반환값 추가
- iOS + 미설치 상태에서 알림 구독 시도 시 "홈 화면에 추가 후 가능" 안내

### 7.7 Service Worker 캐싱
- 정적 에셋: Cache First
- API: Network First (캐시 폴백)

### 7.8 `page.tsx` 통합
- `InstallPrompt` 컴포넌트 연결
