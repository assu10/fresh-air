# Phase 7 설계 — PWA 강화 (모바일 설치 경험 개선)

## 배경 및 목표

Fresh Air는 Next.js PWA로 배포되어 있으나, 사용자가 앱처럼 설치해서 쓰는 경험이 없다.
네이티브 앱 스토어 대신 PWA를 강화해 무료로 Android/iOS에서 홈 화면 설치를 지원한다.

- **Android Chrome**: `beforeinstallprompt` 이벤트로 커스텀 설치 배너 표시
- **iOS Safari**: "홈 화면에 추가" 3단계 안내 모달 제공
- **iOS Web Push**: 홈 화면 설치(standalone) 상태에서만 동작 → 미설치 시 안내

## 제약 조건

| 제약 | 내용 |
|------|------|
| iOS Web Push | iOS 16.4+ Safari + standalone 설치 필수 |
| beforeinstallprompt | Android Chrome 전용 (iOS 미지원) |
| Service Worker | HTTPS 또는 localhost 환경만 |
| 스토어 배포 | 미적용 (무료 방향 — 홈 화면 설치로 대체) |

## useInstallPrompt 훅 설계

```
반환값:
  canInstall     boolean   — 설치 가능 (Android: prompt 있음 / iOS: 미설치)
  isInstalled    boolean   — standalone 모드로 실행 중
  platform       'ios' | 'android' | 'other'
  promptInstall  () => Promise<void>
                   Android: deferredPrompt.prompt() 호출
                   iOS: showIOSGuide = true 설정
  showIOSGuide   boolean   — iOS 안내 모달 표시 여부
  dismissIOSGuide () => void
```

## useNotification 변경

```
기존 반환값: { isSubscribed, isSupported, loading, toggle }
추가 반환값: requiresInstall: boolean

requiresInstall = true 조건:
  - iOS 기기 AND
  - PushManager 지원 AND
  - standalone 모드 아님
```

## InstallPrompt UI 동작

```
공통:
  isInstalled=true OR canInstall=false → null (렌더 안 함)

Android:
  하단 고정 배너 → "앱으로 설치" 버튼 → Chrome 설치 프롬프트

iOS (canInstall=true, showIOSGuide=false):
  하단 고정 배너 → "홈 화면에 추가" 버튼 → showIOSGuide=true

iOS (showIOSGuide=true):
  모달:
    1. Safari 하단 공유 버튼(□↑) 탭
    2. 홈 화면에 추가 탭
    3. 추가 탭
  → 확인 버튼 → dismissIOSGuide()
```

## Service Worker 캐싱 전략

```
설치(install): 정적 에셋 사전 캐싱 ['/', '/icons/icon-192.png', '/icons/icon-512.png']
활성화(activate): 이전 버전 캐시 정리

fetch:
  /api/* 요청  → Network First (최신 데이터 우선, 실패 시 캐시 폴백)
  그 외        → Cache First (오프라인 지원)
```

## TDD 테스트 시나리오

### useInstallPrompt
1. standalone 모드 → `isInstalled=true`
2. iPhone userAgent → `platform='ios'`
3. iPad userAgent → `platform='ios'`
4. Android userAgent → `platform='android'`
5. iOS + 미설치 → `canInstall=true`
6. iOS + 설치됨 → `canInstall=false`
7. iOS에서 promptInstall() → `showIOSGuide=true`
8. dismissIOSGuide() → `showIOSGuide=false`
9. beforeinstallprompt 이벤트 → `canInstall=true`

### InstallPrompt
1. Android + canInstall=true → "앱으로 설치" 버튼 렌더
2. iOS + canInstall=true → "홈 화면에 추가" 버튼 렌더
3. isInstalled=true → null
4. canInstall=false → null
5. 설치 버튼 클릭 → promptInstall() 호출
6. showIOSGuide=true → 안내 모달 표시
7. 확인 버튼 클릭 → dismissIOSGuide() 호출

### useNotification (iOS 추가)
1. 비iOS 기기 → `requiresInstall=false`
2. iOS + 미설치 → `requiresInstall=true`, `isSupported=false`
3. iOS + standalone → `requiresInstall=false`
