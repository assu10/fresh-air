# TASK-20260505 — Phase 5 PWA 프론트엔드 구현

## 작업 목표
미세먼지 환기 알림 PWA의 프론트엔드를 구현한다.

## 완료된 작업

### 5.1 `app/manifest.ts`
- Next.js 16 App Router 방식 PWA manifest
- name: Fresh Air, theme_color: #22C55E, display: standalone

### 5.2 `public/sw.js`
- push 이벤트 → showNotification
- notificationclick → 앱 포커스 또는 새 창

### 5.3 `lib/hooks/useAirQuality.ts`
- coords(lat/lng) 변경 시 /api/air-quality fetch
- loading / data / error 상태 관리

### 5.4 `lib/hooks/useNotification.ts`
- SW 등록 + 기존 구독 여부 확인
- toggle(): 구독 ↔ 해제 + Redis 동기화
- VAPID 키: NEXT_PUBLIC_VAPID_PUBLIC_KEY

### 5.5 `components/LocationDisplay.tsx`
- loading: 스켈레톤 | 정상: 📍 addr

### 5.6 `components/AirQualityCard.tsx`
- loading: 스켈레톤 | error: 에러 메시지 | 정상: 색상 카드

### 5.7 `components/NotificationToggle.tsx`
- isSupported=false: 비활성화 | 토글 버튼

### 5.8 `app/page.tsx`
- 'use client', coords=null → 온보딩, coords≠null → 데이터 화면

### 5.9 `app/layout.tsx`
- metadata 업데이트 (title: Fresh Air, lang: ko)
