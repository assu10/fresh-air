# TASK-20260503 — 프로젝트 초기 설정 (Phase 1)

- [[ADR-20260503-알림방식선택]]

## 작업 목표
Next.js 프로젝트 골격을 생성하고, 프로젝트 문서(CLAUDE.md, PRD.md, WBS.md)와 환경 변수 템플릿을 작성한다.

## 완료된 작업

### 1. Next.js 프로젝트 생성
```bash
npx create-next-app@latest . --typescript --tailwind --app --no-src-dir --import-alias "@/*" --no-git
```
- Next.js 16.2.4, TypeScript, Tailwind CSS, App Router 적용

### 2. 의존성 설치
```bash
npm install web-push @upstash/redis
npm install -D @types/web-push
```

### 3. 생성된 파일 목록

| 파일 | 설명 |
|------|------|
| `CLAUDE.md` | 프로젝트 개요, 구조, 개발 명령어, 주의사항 |
| `PRD.md` | 사용자 스토리, 기능 요구사항, UI 구성, 성공 기준 |
| `WBS.md` | 전체 작업 분류 구조 (Phase 1~6) |
| `.env.local.example` | 환경 변수 템플릿 + 발급 방법 안내 |
| `docs/ADR/` | 아키텍처 결정 기록 디렉터리 |
| `docs/TASK/` | 작업 결과 기록 디렉터리 |
