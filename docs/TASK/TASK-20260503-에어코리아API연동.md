# TASK-20260503 — 에어코리아 API 연동 (Phase 2)

## 연관 ADR
- [[ADR-20260503-에어코리아API연동]]
- [[ADR-20260503-기술스택선택]]
- [[ADR-20260503-PM2.5임계값설정]]

## 작업 목표
브라우저 GPS 좌표를 받아 가장 가까운 에어코리아 측정소의 PM2.5 실시간 데이터를 반환하는 서버 사이드 API를 구현한다.

## 완료된 작업

### 2.1 `lib/geo.ts`
- WGS84 → Korean TM (EPSG:5181) 좌표 변환 (`wgs84ToTm`)
  - GRS80 타원체 기반 Snyder TM 공식 직접 구현 (외부 라이브러리 미사용)
- PM2.5 수치 → 등급/색상/환기가능여부 반환 (`getGradeInfo`)

### 2.2 `lib/airkorea.ts`
- `getNearbyStations(tmX, tmY)`: 근접 측정소 최대 3개 조회
- `getRealtimeAirQuality(stationName)`: 측정소별 PM2.5 실시간 값 조회
- PM2.5 값 `"-"` / `null` 방어 처리
- serviceKey 이중 인코딩 방지

### 2.3 `app/api/air-quality/route.ts`
- `GET /api/air-quality?lat=&lng=`
- 좌표 유효성 검사 (한국 영역 33~39°N, 124~132°E)
- 에어코리아 2단계 API 호출 → 통합 응답 반환

## 생성된 파일

| 파일 | 역할 |
|------|------|
| `lib/geo.ts` | WGS84↔TM 변환, PM2.5 등급 유틸 |
| `lib/airkorea.ts` | 에어코리아 API 클라이언트 |
| `app/api/air-quality/route.ts` | 브라우저용 공기질 프록시 API |

## 검증 방법
에어코리아 API 키 발급 후:
```bash
curl "http://localhost:3000/api/air-quality?lat=37.5&lng=127.0"
```
기대 응답:
```json
{
  "stationName": "강남구",
  "pm25": 18,
  "grade": "보통",
  "canVentilate": true,
  "dataTime": "2026-05-03 14:00"
}
```
