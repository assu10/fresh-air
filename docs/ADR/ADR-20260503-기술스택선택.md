# ADR-20260503 — 기술 스택 선택

## 상태
승인됨

## 배경
미세먼지 개선 시 실시간 푸시 알림을 발송하는 PWA를 만들기 위해 플랫폼, 프레임워크, 인프라를 선택해야 했다.

## 결정

| 레이어 | 선택 | 이유 |
|--------|------|------|
| 플랫폼 | Web App + PWA | iOS/Android 동시 지원, 별도 앱스토어 배포 불필요 |
| 프레임워크 | Next.js (App Router, TypeScript) | SSR + API Routes 통합, Vercel 최적화 |
| 스타일링 | Tailwind CSS | 빠른 UI 구성 |
| 배포 | Vercel Hobby (무료) | Next.js 최적 호스팅, 별도 서버 관리 불필요 |
| 저장소 | Upstash Redis (무료 tier) | Serverless 환경에 적합한 HTTP 기반 Redis |
| 크론 | cron-job.org | Vercel Hobby 크론은 하루 1회 제한 → 외부 무료 크론으로 30분 간격 가능 |
| 공기질 데이터 | 에어코리아 API (공공데이터포털) | 한국 공식 기관 데이터, 무료, 측정소 단위 조회 가능 |

## 검토한 대안

- **Firebase**: Hosting + Functions + FCM 통합 가능하나, 설정 복잡도 높고 Vercel과 비교해 Next.js 친화성 낮음
- **자체 서버(VPS)**: 최대 제어권이지만 서버 관리 부담, 비용 발생
- **클라이언트 폴링만**: 백엔드 불필요하나 앱 종료 시 알림 불가 → 핵심 요구사항 미충족

## 결과
모든 인프라를 무료로 운영하면서 백그라운드 푸시 알림 요구사항을 충족하는 스택 확정.
