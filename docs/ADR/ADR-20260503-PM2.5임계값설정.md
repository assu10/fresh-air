# ADR-20260503 — PM2.5 임계값 설정

## 상태
승인됨

## 배경
"환기해도 좋은" 기준을 PM2.5 수치로 정의해야 했다. 한국 환경부 등급 기준을 참고했다.

## 결정
**PM2.5 ≤ 35 μg/m³** ('좋음' + '보통' 등급)을 환기 가능 기준으로 채택.

## 한국 환경부 PM2.5 등급 기준

| 등급 | 범위 | 의미 |
|------|------|------|
| 좋음 | 0–15 μg/m³ | 환기 적극 권장 |
| 보통 | 16–35 μg/m³ | 환기 가능 |
| 나쁨 | 36–75 μg/m³ | 환기 자제 |
| 매우나쁨 | 76+ μg/m³ | 환기 금지 |

## 모니터링 항목
- PM2.5만 측정 (PM10 제외)
- 인체에 더 유해한 초미세먼지(PM2.5)를 기준으로 삼는 것이 적절

## 검토한 대안
- **'좋음' 등급만(≤15)**: 너무 엄격해 알림 빈도 낮음
- **사용자 직접 설정**: MVP 범위 초과, 추후 확장 기능으로 분류
