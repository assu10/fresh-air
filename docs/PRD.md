# PRD — Fresh Air (미세먼지 환기 알림 PWA)

## 배경 및 목적

실내 환기를 하려고 해도 밖의 미세먼지 상태를 매번 확인하기가 번거롭다. 미세먼지가 나쁜 상태에서 환기를 하면 오히려 실내 공기질이 나빠질 수 있다. 이 앱은 사용자 대신 미세먼지를 모니터링하다가 환기해도 좋은 시점이 되면 즉시 알림을 보내준다.

## 핵심 사용자 스토리

- 앱에 접속하면 현재 내 위치의 PM2.5 수치와 등급을 바로 확인할 수 있다.
- "환기 알림 받기"를 켜두면, 앱을 닫은 상태에서도 미세먼지가 개선되는 순간 푸시 알림을 받는다.
- 알림을 받으면 "지금 창문 열어도 된다"는 것을 즉시 알 수 있다.

## 기능 요구사항

### 필수 (MVP)

| # | 기능 | 설명 |
|---|------|------|
| F1 | 현재 공기질 표시 | GPS로 현재 위치 감지 → 가장 가까운 에어코리아 측정소 PM2.5 수치 및 등급 표시 |
| F2 | 자동 위치 감지 | 브라우저 Geolocation API 사용, 권한 요청 후 최근접 측정소 자동 선택 |
| F3 | 푸시 알림 구독 | Web Push API로 구독 등록/해제. 구독 정보는 서버에 저장 |
| F4 | 백그라운드 알림 | 앱 미사용 중에도 PM2.5가 35 μg/m³ 이하로 개선되면 즉시 알림 발송 |
| F5 | 임계값 초과→이하 전환 감지 | 이전 수치 > 35 AND 현재 수치 ≤ 35일 때만 알림 발송 (중복 알림 방지) |
| F6 | PWA 설치 | 홈 화면에 앱 추가 가능 (manifest.json + Service Worker) |

### 제외 (MVP 범위 밖)

- PM10 모니터링
- 사용자별 임계값 커스터마이징
- 알림 수신 시간대 설정
- 여러 지역 즐겨찾기
- 과거 데이터 차트

## 비기능 요구사항

- **비용**: 모든 인프라 무료 플랜으로 운영 가능해야 함
- **반응성**: 미세먼지 개선 후 최대 30분 내 알림 도달
- **정확성**: 에어코리아 공식 데이터 사용 (한국환경공단)
- **보안**: API 키는 서버 사이드에서만 사용, 크론 엔드포인트는 secret으로 보호

## 알림 기준

| 조건 | 동작 |
|------|------|
| PM2.5 > 35 μg/m³ (나쁨/매우나쁨) | 알림 없음 |
| PM2.5 ≤ 35 μg/m³으로 개선됨 | 🌿 "지금 환기하기 좋아요!" 알림 발송 |
| 이미 좋은 상태에서 계속 좋음 | 알림 없음 (중복 방지) |

## UI 구성 (단일 화면)

```
┌─────────────────────────────┐
│  📍 서울 강남구              │
│                             │
│    ●  18 μg/m³              │
│       보통 — 환기 가능       │
│                             │
│  [🔔 환기 알림 받기] ON/OFF │
│                             │
│  마지막 업데이트: 14:30      │
└─────────────────────────────┘
```

- PM2.5 등급에 따라 원형 인디케이터 색상 변경 (파란/초록/주황/빨간)
- 알림 구독 중이면 토글이 활성화 상태로 표시

## 기술 스택 요약

| 항목 | 선택 |
|------|------|
| 플랫폼 | Web App + PWA |
| 프레임워크 | Next.js (App Router, TypeScript) |
| 공기질 데이터 | 에어코리아 API (공공데이터포털) |
| 알림 인프라 | Web Push Protocol + VAPID |
| 저장소 | Upstash Redis |
| 배포 | Vercel |
| 크론 | cron-job.org (30분 간격) |

## 외부 서비스 의존성

| 서비스 | 용도 | 비용 |
|--------|------|------|
| 공공데이터포털 (data.go.kr) | 에어코리아 API 키 발급 | 무료 |
| Upstash | Redis (푸시 구독 저장) | 무료 tier |
| Vercel | 호스팅 | 무료 Hobby 플랜 |
| cron-job.org | 30분 간격 HTTP 크론 | 무료 |

## 성공 기준

- 미세먼지가 35 μg/m³ 이하로 개선됐을 때 30분 이내에 알림 도달
- 앱을 닫은 상태(백그라운드)에서도 알림 수신 가능
- 모바일 브라우저에서 홈 화면 추가(PWA 설치) 가능
